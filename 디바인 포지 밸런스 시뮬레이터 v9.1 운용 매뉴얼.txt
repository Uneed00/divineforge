 디바인 포지 밸런스 시뮬레이터 v9.1 운용 매뉴얼
1. 제어 패널 (Control Panel) - 기본 설정
시뮬레이션의 기본 환경을 설정하는 구역입니다.

시뮬레이션 횟수: 100 / 1,000 / 10,000회 중 선택합니다.

Tip: 단순 테스트는 1,000회, 밸런스 확정용 데이터는 10,000회를 권장합니다. (오차 범위 ±0.5% 이내)

신화 배정 (P1~P4): 4명의 플레이어가 선호할 '주력 신화(Target Myth)'를 지정합니다.

없음(무가중치): AI가 상황에 따라 유동적으로 판단합니다.

특정 신화 지정 시: 해당 AI는 게임 초반부터 해당 신화의 유물을 최우선으로 수집하려 시도합니다. (특정 신화 간의 상성 테스트용)

버튼 기능:

▶ 추가 실행: 현재 통계에 데이터를 누적하여 시뮬레이션을 실행합니다.

⏹ 중지: 실행 중인 시뮬레이션을 즉시 멈춥니다.

🔄 통계 초기화: 누적된 모든 데이터를 삭제하고 0부터 다시 시작합니다.

2. 밸런스 파라미터 미세 조정 (Advanced Settings)
⚙️ 밸런스 파라미터 미세 조정 탭을 클릭하면 열립니다. 기획 의도에 맞춰 수치를 수정하여 테스트할 수 있습니다.

A. ☘️ 켈트 (Celt) - 변칙 운영
미믹(3단계) 활성: 체크 시, 켈트 유물 3개 이상 보유 시 다른 신화의 능력을 복사하는 '미믹' 로직이 작동합니다.

교환 횟수 (0~3단계): 쉼표(,)로 구분하여 입력. 예: 0,1,2,3 (유물 개수에 따른 자원 교환 가능 횟수)

변환(Convert) 발동 레벨: 숫자 하나만 입력합니다. 해당 레벨 이상일 때 '축복 없음' 페널티를 자원 교환으로 무마하는 로직이 활성화됩니다. (예: 2 입력 시 2개부터 발동)

면제(Free) 발동 레벨: 숫자 하나만 입력합니다. 해당 레벨 이상일 때 축복을 아예 소모하지 않습니다. (예: 3 입력 시 3개부터 발동)

B. ☁️ 올림포스 (Olympus) - 자원 할인
단계별 할인량: 쉼표로 구분. 예: 0,2,5,12 (0개, 1개, 2개, 3개 보유 시 각각 할인되는 자원 수치)

C. 🌲 노르드 (Nord) - 핸드 제한 (Hand Limit)
Lv2 / Lv3 핸드 제한: 노르드 유물이 2개(Silver), 3개(Gold) 이상일 때 적용될 최대 손패 장수를 입력합니다. 기본값은 8장이며, 이 수치를 늘려 노르드의 3장 콤보 수집(Multi-Gather) 효율을 높일 수 있습니다.

D. 👁️ 이집트 (Egypt) - 행동 비용 감소
Matrix 입력: LV1 ~ LV3 각 단계 도달 시 추가로 감소할 행동력(Time)을 설정합니다.

예약(Reserve): 유물 찜하기 비용

자원(Res): 자원 가져오기 비용

축복(Bless): 축복 카드 가져오기 비용

제작(Craft): 유물 제작 비용

3. 대시보드 해석 가이드 (Analytics)
시뮬레이션 완료 후 출력되는 데이터의 해석법입니다.

① ⚡ 신화별 성능 분석 (Ratio v2)
가장 중요한 메인 지표입니다. 각 신화 덱의 종합 성능을 보여줍니다.

신화 (Tier): 해당 덱의 성능 등급입니다.

OP (Tier 0): 밸런스 붕괴 수준의 강력함 (Power Score 150↑)

Strong (Tier 1): 강력함

Balanced: 적절함

Weak / Under: 상향 필요

승률 (Win Rate): 4인 게임 기준, 기대 승률인 25%를 기준으로 얼마나 높은지 보여줍니다.

σ (Sigma): 표준편차 대비 승률 격차입니다. +수치가 높을수록 강력합니다.

Depth (t2/t3): 승리했을 때 유물을 '2개(t2)' 모으고 이겼는지, '3개 이상(t3)' 모으고 이겼는지 비율입니다. t3 비율이 높으면 '왕귀형(성장형)' 덱, t2 비율이 높으면 '속공형' 덱입니다.

평균 순위 (Rank): 1~4위 중 평균적으로 몇 위를 기록했는지 보여줍니다. (낮을수록 좋음, 2.0 미만이면 매우 강력)

고점 (Elite Win Rate): 상위 5%의 고득점 게임에서 해당 신화가 승리한 비율입니다. '숙련자용 메타'에서의 잠재력을 의미합니다.

Score (Power Score): 승률, 순위 방어율, 고점을 종합한 최종 밸런스 점수입니다. 100~120점 사이가 황금 밸런스입니다.

② 📅 신화별 성장 곡선 (일일 평균 점수)
게임 내 1일차~5일차(종료)까지의 점수 획득 추이입니다.

붉은색 (+수치): 전날 대비 점수가 폭발적으로 상승했음을 의미합니다. (엔진 가동 시점 파악 용도)

③ 🏆 Top 5 고득점 덱 분석
이번 시뮬레이션에서 나온 가장 높은 점수의 게임 5개를 보여줍니다.

덱 구성: 승자가 어떤 유물을 몇 개씩 섞어 썼는지 보여줍니다. (예: 올림 3, 이집 1) -> 혼합(Mix) 전략의 유효성을 판단할 때 봅니다.

④ 📈 플레이어 승률 & 💰 경제 효율성
P1~P4 승률: 턴 순서(선플레이어/후플레이어)에 따른 유불리를 확인합니다. P1이 압도적으로 높다면 선공 필승 밸런스입니다.

자원 소모 효율: (소모한 자원 / 획득한 자원) %. 100%에 가까울수록 낭비 없이 딱 맞춰 썼다는 뜻입니다. 보통 85% 이상이면 고효율입니다.

1-2위 평균 격차: 1등과 2등의 점수 차이입니다. 이 수치가 너무 크면(20점 이상) 게임이 일방적이라는 뜻입니다.

⑤ 🛠️ 특수 능력 활용 (평균)
각 신화의 고유 능력이 게임당 평균 몇 번 발동되었는지 체크합니다.

올림포스: 자원 절약 수치 (높을수록 좋음)

노르드: 버려진 카드 수 (콤보 수집을 위해 패를 얼마나 순환시켰는가)

켈트: 자원 변환 횟수 (유연성 지표)

4. 개발자 팁 (업무 활용)
OP 판독: '신화별 성능 분석'에서 Power Score가 다른 신화보다 30점 이상 높거나, Tier Label이 빨간색(OP)이면 즉시 너프(Nerf)가 필요합니다.

안정성 테스트: 승률은 낮지만 평균 순위가 높은 신화(예: 2.2위)는 2등을 꾸준히 하는 '순위 방어형' 덱입니다. 이는 대회나 리그전에서 중요한 지표가 됩니다.

데이터 저장: 우측 상단의 📥 Raw CSV 버튼을 누르면 엑셀로 가공 가능한 원본 데이터를 다운로드할 수 있습니다. 심층 분석 시 활용하십시오.
